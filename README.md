# Destroy the aliens

The library called pygame was used in this project for my first game made with python.

# How to start?

In a console or terminal open the file named main.py

# Game controls

- Left arrow
- Right arrow
- Up arrow
- Arrow down
- Space (shoot)
- Q (Exit game)
