## üéÆ About the Game

**Alien Invasion** is a modern take on the classic arcade shooter, where you
pilot a spaceship to defend Earth from waves of alien invaders. With stunning
visuals, immersive sound effects, and addictive gameplay, it's a perfect blend
of nostalgia and modern gaming.

### üåü Key Features

- üöÄ **Fast-Paced Action**: Quick reflexes and strategic shooting
- üëΩ **Dynamic Enemies**: Multiple alien types with unique behaviors
- üí• **Explosive Combat**: Satisfying particle effects and animations
- üéµ **Immersive Audio**: Dynamic sound effects and background music
- üíØ **High Score System**: Track your best performances
- üé® **Retro-Modern Style**: Classic arcade feel with modern polish
- üåç **Multilingual Support**:
    - Automatic language detection
    - English and Spanish support
    - More languages coming soon
- üéØ **Visual Enhancements**:
    - High-contrast UI with semi-transparent overlays
    - Smooth animations and transitions
    - Clear game information display
