# Code Destroy the aliens

## Start developing

1.- Create a virtual environment

```python
python -m venv env
```

2.- Activate virtual environment

```bash
env\Scripts\Activate
```

3.- Install packages

```bash
pip install -r requirements.txt
```

## Game controls

- Left arrow
- Right arrow
- Up arrow
- Down arrow
- Space (shoot)
- Q (Exit game)

## libraries and modules used in the project

- Time
- Sys
- Pygame
- Pygame.sprite
- Pygame.font
