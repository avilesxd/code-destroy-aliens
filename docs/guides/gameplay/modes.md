Alien Invasion currently offers a single classic mode focused on arcade-style

## Classic Mode

- Waves of aliens descend from the top

- Difficulty increases each level
- High score is tracked and saved

## Future Modes

Additional modes (survival, challenge, multiplayer) are not yet implemented. If
you want to contribute new modes, start by reviewing the game loop and
statistics systems.
