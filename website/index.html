System Requirements

Minimum

  • OS: Windows 10 64-bit
  • Processor: Dual-core CPU @ 2.0 GHz
  • Memory: 1 GB RAM
  • Graphics: Integrated graphics with OpenGL support
  • Storage: 1 GB available space
  • Network: Not required for single player

Recommended

  • OS: Windows 10/11 64-bit
  • Processor: Any modern dual-core CPU
  • Memory: 2 GB RAM
  • Graphics: Any graphics card with OpenGL support
  • Storage: 2 GB available space
  • Network: Not required for single player
  • Monitor: 1280x720 minimum resolution

* The game is very lightweight and runs well on most modern computers. It only requires Windows 10 or later and a basic graphics card with OpenGL support.
